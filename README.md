# Telegram OSINT Toolkit

**High-end local Telegram intelligence tool** for OSINT, cybersecurity research, law enforcement, and military intelligence purposes.

Built with **Telethon** — supports proxy/Tor, bulk monitoring, advanced search, exports, and more.

> **⚠️ Legal Notice**: Use responsibly and only on data you are authorized to access. Respect Telegram ToS and local laws.

## Features
- Proxy / Tor support (SOCKS5, HTTP)
- Bulk keyword search & monitoring from channel list
- Full chat export (JSON/CSV) with optional media download
- Member enumeration & user profiling
- SQLite database for persistent storage
- Rate-limit friendly with delays
- Rich CLI interface
